# AB-test-statistical-analysis
Статистичний аналіз результатів A/B тесту та перевірка гіпотез

### Опис проєкту
Метою проєкту було оцінити вплив тригера «знижка 50%» на конверсію в покупку. У ході роботи було проведено порівняльний аналіз контрольної (A) та тестової (B) груп для прийняття рішення про впровадження змін.

### Інструменти та технології
* **Python (Pandas, SciPy, Statsmodels):** проведення статистичних тестів.
* **Matplotlib / Seaborn:** візуалізація розподілів та довірчих інтервалів.
* **Статистичні методи:** перевірка на нормальність розподілу, розрахунок p-value, оцінка статистичної значущості.

### Що було зроблено:
1. **Дизайн тесту:** перевірка груп на репрезентативність та відсутність перетину аудиторій.
2. **Exploratory Data Analysis (EDA):** аналіз викидів та аномалій у поведінці користувачів.
3. **Статистична перевірка:** застосування комплексу тестів (Z-test, Welch’s T test та Xi-square) для перевірки нульової гіпотези.
4. **Розрахунок потужності:** оцінка ймовірності помилок I та II роду.

### 💡 Ключовий висновок
За результатами аналізу було виявлено статистично значущу різницю між варіантами. Сформовано рекомендації для бізнесу щодо подальшого розгортання оновлення.
